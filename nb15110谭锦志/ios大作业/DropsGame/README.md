##游戏概况

- 基于cocos2d-x 3.9

- 发布平台：Android、iOS9

##游戏规则

- 初始状态玩家拥有10滴水滴

- 填补游戏区水滴直至水滴爆炸，获得连锁反应

- 所有水滴均消去则获胜

- 未能消去所有所有水滴则失败
