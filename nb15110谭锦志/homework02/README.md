##iOS第二次作业

* 使用make构建
* 实现功能：
	* 编译：
	```cpp
    	make
    ```
    * 运行：
    ```cpp
    	./MyCal           // 输出当月日历
        ./MyCal 10 2014   // 输出指定年月日历
        ./MyCal -m 10     // 输出当年10月日历
        ./MyCal 2014      // 输出指定年份日历
    ```
    * 卸除
    ```cpp
    	make clean
    ```
	* 参数限制:
		* 最多允许两个参数
		* 只有一个参数时默认为年份，范围`1-9999`
		* 有两个参数时格式: `month year`,月份范围：`1-12`
